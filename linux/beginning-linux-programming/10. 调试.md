# 10.2 常用调试技巧

程序调试可以分为如下5个阶段: 

1. 测试：找出程序中存在的缺陷或错误
2. 固化：使错误可重现
3. 定位：确定相关的代码行
4. 纠正：修改代码纠正错误
5. 验证：确定修改解决了问题
