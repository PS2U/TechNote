# 计算机硬件介绍

- **CPU**：取指、解码、执行。另有一些通用寄存器、程序计数器、堆栈指针、程序状态字。流水线、超标量CPU
- **存储器**：寄存器、高速缓存、主存（RAM）、磁盘、磁带
- **IO设备**：控制器和设备本身，专门去控制器对话的软件叫做设备驱动，IO设备的输入输出有三种方式：查询、中断、DMA
- **总线**：高速缓存总线、局部总线、内存总线、PCI总线、ISA总线、USB、SCSI、IEEE 1394
